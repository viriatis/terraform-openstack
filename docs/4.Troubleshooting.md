# Troubleshooting

## Bootstrap issues 

### Disk is not empty
```shell
ubuntu@ubuntu-server:~$ sunbeam cluster bootstrap  --role control,compute,storage
Ceph devices (/dev/disk/by-id/ata-EDILOCA_ES106_4TB_AA00000000010276): /dev/sda
An unexpected error has occurred. Please see https://canonical-openstack.readthedocs-hosted.com/en/latest/how-to/troubleshooting/inspecting-the-cluster/ for troubleshooting information.
Error: task error: Task 32: status 'failed', return code 0, details:
Results: {'result': "[{'spec': '/dev/sda', 'status': 'failure', 'message': 'Error: data device /dev/disk/by-id/ata-EDILOCA_ES106_4TB_AA00000000010276 is not pristine (contains data) - use --wipe to override\\n'}]"}
```

**Solution:**
```shell
ubuntu@ubuntu-server:~$ sudo microceph disk add /dev/sda --wipe
```
