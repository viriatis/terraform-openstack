### How to change endpoints

```shell
openstack endpoint set <id> --url 'http://<ip or hostname>/<uri>'

openstack endpoint set 8bac982a8e0c42b3a2e0690ee1c3ffc4 --url http://10.57.123.1
openstack endpoint set 495c6482b51f48e788781addfb532568 --url 'http://10.57.123.1/swift/v1/AUTH_$(project_id)s' 
openstack endpoint set bfdd5497e9cb4fe299d4feab3a97a748 --url http://192.168.200.102:3000
openstack endpoint set 45e0514c37f245d89e96b5da0bcd2b43 --url 'http://192.168.200.102:3000/swift/v1/AUTH_$(project_id)s'
```
