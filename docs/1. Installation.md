# Instalation

## Install Openstack
[Architecture-openstack](images/architecture-openstack.png)

### Install openstack with snapd
```shell
sudo snap install openstack
```

### Prepare machine

Before we’ll be able to bootstrap the cloud, we have to prepare the machine for Canonical OpenStack usage

```shell
sunbeam prepare-node-script --bootstrap | bash -x && newgrp snap_daemon
```

### Bootstrap the cloud
You can use the flag '--accept-defaults' for a default automatic bootstrap, but I prefer without it to control and understand what is happening and being configured.

```shell
sunbeam cluster bootstrap  --role control,compute,storage
```

Check in another shell session the installion. You might see a lot of warnings or errors that are just a matter of time to be solved. To checkout the status of the installation, use the following commands:

```shell
$ juju status # you might need to do 'juju switch <<machine>>' to be able to see it's status

$ sudo k8s status # the openstack cloud is based on a k8s cluster
$ sudo k8s kubectl get nodes -A
$ sudo micro
```
