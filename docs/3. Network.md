# Sunbeam — br-ex Bridge & Floating IP Setup

## Create br-ex via netplan

Create `/etc/netplan/99-br-ex.yaml`:

```yaml
network:
  version: 2
  renderer: networkd
  ethernets:
    enp3s0:
      dhcp4: false
  bridges:
    br-ex:
      interfaces: [enp3s0]
      dhcp4: true
      parameters:
        stp: false
        forward-delay: 0
```

Apply:

```bash
sudo netplan apply
```

Verify:

```bash
ip addr show br-ex
# Should show 192.168.200.102/24 on br-ex
```

## Tell Sunbeam to use br-ex

The `openstack-hypervisor` Juju charm manages OVN. Configure it:

```bash
juju config openstack-hypervisor -m sunbeam-controller:admin/openstack-machines \
  external-bridge=br-ex \
  external-bridge-address=192.168.200.102/24
```

This tells OVN to use `br-ex` as the provider bridge for external network traffic.


## Verify OVN wiring

```bash
sudo openstack-hypervisor.ovs-vsctl show
```

Expected output shows:
- `br-ex` bridge with `enp3s0` as a port
- `patch-provnet-*` connecting `br-ex` to `br-int`
